Versionamento

o git é um software que permite realizar versões de um projeto. O github desktop é um software que faz a ligação entre visual studio code, git e github

github desktop permite fazer a sincronia entre o repositório local e o rpositório remoto

commit

é o processo de confirmação de uma versão, ou seja, a cada operação é possivel confirma a alteração

Push

esse comando envia (sincroniza) as alteraçoes realizadas mo repositorio local com o repositorio remoto
